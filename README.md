# Enhanced Skin Cancer Detection & Classification

AI detects skin cancer (Melanoma, BCC, SCC) using InceptionV3/MobileNet on ISIC data. Achieving 97.8% accuracy, it provides clinical treatment advice and urgent doctor referral triggers.

## 🚀 Features
- **Deep Learning:** MobileNet & InceptionV3 Architectures.
- **Decision Support:** Automated treatment suggestions.
- **Urgency Logic:** Direct "Doctor Meet" referral triggers.

## 🛠️ Installation
1. Clone the repo: `git clone <your-url>`
2. Install dependencies: `pip install -r requirements.txt`
3. Run app: `streamlit run app.py`

## 📊 Dataset
Trained on the **ISIC (International Skin Imaging Collaboration)** dataset.
